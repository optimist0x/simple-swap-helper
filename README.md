# simple-swap-helper

Классическая игра Змейка на HTML5 Canvas и JavaScript.

## Особенности
- Плавное управление
- Система счета
- Пауза и перезапуск
- Адаптивный дизайн

## Управление
- Стрелки: движение змейки
- Кнопка "Пауза": приостановить игру
- Кнопка "Начать игру": новая игра

## Цель
Собирайте красную еду, чтобы расти и набирать очки!
