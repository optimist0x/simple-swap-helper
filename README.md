## Features

- **Smooth Controls** - Responsive arrow key controls
- **Score Tracking** - Real-time score tracking system
- **Pause Functionality** - Pause and resume gameplay
- **Responsive Design** - Works on all screen sizes
- **Modern UI** - Clean and beautiful interface

## Controls

| Key | Action |
|-----|--------|
| Arrow Up | Move up |
| Arrow Down | Move down |
| Arrow Left | Move left |
| Arrow Right | Move right |
| Start Game Button | Begin a new game |
| Pause Button | Pause/resume current game |

## Gameplay

- Collect red food to grow longer and increase your score
- Avoid hitting walls or the snake will die
- Don't hit yourself - the game ends if you collide with your own body
- Speed increases as your score grows

## Installation

1. Clone this repository:
```bash
git clone https://github.com/optimist0x/simple-swap-helper.git
```

2. Open `index.html` in your browser:
```bash
cd simple-swap-helper
open index.html  # macOS
# or just double-click index.html
```
